# Repositorios del Grado en Diseño y Desarrollo de Videojuegos

Bienvenidos 👋

Este espacio reúne repositorios de apoyo para las asignaturas del [Grado en Diseño y Desarrollo de Videojuegos](https://www.unir.net/diseno/grado-diseno-desarrollo-videojuegos/) de UNIR. El objetivo es ofrecer material complementario y práctico: ejemplos, plantillas, ejercicios guiados, conjuntos de datos, recursos de apoyo y proyectos de referencia para ayudarte en el desarrollo de algunas clases y actividades.

🎯 Qué encontrarás aquí

Según la asignatura, en cada repositorio, podrás encontrar:

* Ejemplos de código (proyectos mínimos, mecánicas concretas, patrones de arquitectura, utilidades).
* Plantillas de proyectos (estructuras recomendadas, convenciones, configuración base).
* Actividades guiadas (paso a paso) y retos para practicar.
* Recursos: lecturas, enlaces, herramientas, assets de prueba (cuando aplique) y documentación breve.
* Material para la evaluación formativa: rúbricas de referencia, criterios y pautas de entrega (si el docente lo indica).

✅ Requisitos recomendados

Cada repositorio indicará sus requisitos concretos. En general, te será útil tener:

* Git instalado y una cuenta de GitHub
* Un editor/IDE (p. ej. Visual Studio Code, Rider)
* Motores/herramientas según la asignatura (por ejemplo, Unity o Unreal Engine, cuando corresponda)

🧠 Integridad académica

Este material está pensado para aprender. Si se publican soluciones, su uso será el que indique el docente. En particular:

* Evita presentar como propio el material que no hayas desarrollado tú
* Cita fuentes externas (código, tutoriales, assets) cuando corresponda
* Si tienes dudas sobre qué está permitido, pregunta al docente antes de entregar

🧾 Licencia y uso de materiales

Cada repositorio puede tener su propia licencia y reglas de uso (por ejemplo, para código, documentación o assets). Revisa el fichero `LICENSE` de cada repo.
Si no hay licencia indicada, asume un uso académico y consulta al docente.
